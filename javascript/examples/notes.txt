How to define function in javascript

function is a keyword in javascript, to define function it is required

function functionname([arguments/para])
{
//function body
[return]
}

content with [] is optional part

function add(x,y)
{
z=x+y;
return z;
}


a=5;
b=6;
c=add(a,b);	//11

a="4"
b=3;	
c=add(a,b);	//"43"
